# easing 함수 계산식

```
t: 현재시간 : time/frame
b: 처음 값  ex) start position = -320 , b = -320
c: 이동해야할 거리. ex 0 -> 100 c= 100
d: 물체가 10-> 40까지 움직일 동안의 총 시간 ex) 60/23 초
```

- LinearTween (일정하게 들어오기)

```js
function linearTween(t, b, c, d) {
  return (c * t) / d + b;
}
```
